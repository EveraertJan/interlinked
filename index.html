Interlinked

Map the individual interactions in your experience. You can either create a graph per interaction, or model them all in one overview.

Where to start

  1. Target a feeling. What emotion should the interaction evoke? Start there, not with the technology.
  2. Pick an Output Effect. What sensory output fits that feeling? FOr example: dim, cold light and a slow soundtrack for something somber. Add the nodes necessary.
  3. Define an Input Effect. What triggers it? Presence in the room, a gesture, a noise (sound level), proximity?
  4. Add a Manipulation node. How does raw input become a trigger. When does the output effect come into play?
  5. Add Input Technology. The sensors or devices that capture the input.
  6. Annotate your nodes. A short label like "PAR can" or a note explaining a design decision keeps the graph readable when sharing.
  7. Add Output Technology. The physical devices that produce that effect. Label the node; a PAR fixture, a speaker, a fog machine.

Controls

Add a node
Double-click or long-press on the canvas
Connect nodes
Drag from the left or right edge of a node
Move a node
Drag from its centre
Pan
Drag on empty canvas, or hold Space + drag
Zoom
Scroll wheel or pinch
Node options
Long-press (or right-click) a node to rename, annotate, or delete

Saving & sharing

Save
Downloads the graph as a .json file — keep it as your working file
Load
Opens a previously saved .json file to continue where you left off
SVG / PNG
Exports a clean image of the graph for sharing in presentations or documents. Annotated nodes include a numbered caption block below the diagram.

Using the auto-layout before exporting creates a more consistent graph. This feature will group graphs based on connections.
